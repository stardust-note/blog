# 그림 강습을 받고 있어요. 어반 아뜰리에 라는 곳에서요.

모던 UI 디자인은 단순함과 일관성을 중심으로 한 디자인 철학이에요.  
오늘날의 웹과 모바일 인터페이스는 **불필요한 장식보다 기능적 미니멀리즘**을 중시합니다.

## 🎨 주요 원칙
<br />
1. **단순함 (Simplicity)**  
   → 복잡한 요소보다는 명확한 정보 전달이 핵심이에요.
<br />
2. **일관성 (Consistency)**  
   → 버튼, 폰트, 간격, 색상 등은 전체 프로젝트에서 통일되어야 합니다.

3. **가독성 (Readability)**  
   → 충분한 여백과 대비는 사용자의 피로를 줄입니다.
